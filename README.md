Using GitHub Pages to host site
Go to "https://cocraighead.github.io/" to view site

To Start
`setup angular`
`setup node modules`

To Run
`cd ".\cocraighead.github.io\site\src\app"`
`ng serve`

To deploy
`cd ".\cocraighead.github.io"`
`node deploy_assets.js`
`cd ".\cocraighead.github.io\site\src\app"`
`ng build`
